# merge_sort
  - C Language
  - in Linux x86_64 (Ubuntu 16.04 LTS)
  - Number of Processor : 4 // Memory 4GB (VMware)
  - Compile with Makefile
  
# how to use
  - git clone https://github.com/zanywhale/merge_sort.git
  - cd merge_sort
  - make
 
Example
```sh
zanywhale@ubuntu:~/git/merge_sort$ ./MergeSort 10000000
```
